Linux shell 编程基础

发表于 2017-03-21 | 分类于 linux |

简单记录一些基础语法

特殊变量

字符串

注意变量定义=两边 不要有空格！！
#一般用来表示长度，比如 str=outyua，可以用${[#str]}来获取str的长度(去掉中括号)

把执行结果保存到变量中

数组

表达式

表达式和运算符之间要有空格，例如 2+2 是不对的，必须写成 2 + 2
完整的表达式要被 ` 包含，注意这个字符不是常用的单引号，在 Esc 键下边。

关系表达式

-eq相等 -ne不等 -gt大于 -lt小于 -ge大于等于 -le小于等于
-a并且 -o或者
-z字符串长度为0 -n长度不是0
直接放到if条件里判断是否为空
-r是否可读 -w是否可写 -x是否可执行 -f是否是普通文件 -d是否是目录 -s文件大小是否为0 -e文件是否存在
两条命令希望第一条执行正确则继续执行：&&
两条命令希望第一条执行错误则继续执行：||
检查一个目录是否存在，并执行操作，类似PHP的?:操作符
test -e /test && echo "exist" || echo "Not exist"

if

case语句

循环语句

break语句会跳出循环break 2会跳出两层循环
continue语句会跳过循环continue 2会跳过两层循环
